## Critério

Uma entrega somente é classificada como concluída quando o código correspondente está presente na linha principal, possui teste automatizado aplicável, documentação da decisão e validação pelo CI. Itens descritos apenas como “próximo bloco” não são considerados entregues pela PR original; eles são rastreados até a PR posterior que os implementou.

## Regra permanente

Nenhuma PR futura será apresentada como concluída enquanto houver item do próprio escopo marcado como próximo bloco, pendente, TODO ou sem teste aplicável. Uma nova PR só será aberta após a anterior estar integralmente validada e mergeada.
